# Marcador de Baloncesto en Kotlin para Android

Este es un proyecto de Android que implementa un marcador de baloncesto interactivo, desarrollado en Kotlin. La aplicación permite a los usuarios seleccionar dos equipos y hacer un seguimiento de su puntuación en tiempo real durante un partido de baloncesto. Es una herramienta útil para registrar y mostrar el marcador de los partidos de baloncesto de forma simple y visual.

## Características

- **Selección de equipos**: Puedes elegir los equipos que participan en el partido.
- **Marcador en tiempo real**: Los puntos de ambos equipos se actualizan dinámicamente durante el partido.
- **Interfaz simple y fácil de usar**: La interfaz está diseñada para ser intuitiva y rápida de usar durante un partido.

## Tecnologías utilizadas

- **Kotlin**: Lenguaje de programación utilizado para desarrollar la aplicación.
- **Android Studio**: Entorno de desarrollo integrado (IDE) para crear aplicaciones Android.
- **XML**: Utilizado para diseñar la interfaz de usuario de la aplicación.
.

## Funcionalidad

1. **Seleccionar equipos**:
   Los usuarios pueden seleccionar los equipos de una lista predefinida .
   
2. **Actualizar marcador**:
   Los puntos de cada equipo se pueden incrementar de manera manual mediante botones, y el marcador se actualizará en tiempo real.

3. **Reinicio del marcador**:
   Al finalizar el partido, puedes reiniciar el marcador para comenzar un nuevo juego.
